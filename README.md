# Calculadora JS

Projeto simples de calculadora feito com **HTML, CSS e JavaScript**.

---

##  Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de erro (divisão por zero)

---

## Tecnologias usadas

- HTML
- CSS
- JavaScript

---

 ├── style.css
